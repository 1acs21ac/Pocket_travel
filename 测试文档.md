# 口袋旅行搭子 - 测试文档

## 一、测试环境配置

### 1.1 前置条件
- 微信开发者工具（最新版）
- 后端服务已启动（`npm run dev`，端口 3000）
- MySQL 数据库已初始化
- 测试微信号（微信开发者工具模拟器或真机）

### 1.2 接口地址配置
```
开发环境接口地址：http://127.0.0.1:3000
```
微信开发者工具中开启「不校验合法域名、web-view（业务域名）」选项。
